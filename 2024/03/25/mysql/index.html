mysql指令

mysql的打开与停止

mysql的打开：

net start mysql80

mysql的停止：

net stop mysql80

mysql客户端的链接：

mysql [-h 127.0.0.1] [-p 3306] -u root -p

SQL

SQL通用语法

  1. SQL语句可以单行或者多行书写，以分号结尾
  2. SQL语句可以使用空格/缩进来增强语句的可读性
  3. MySQL数据库的SQL语句不区分大小写，关键词建议用大写
  4. 注释：
    • 单行注释：–注释内容 或者 #注释内容
    • 多行注释： /* 注释内容 */

SQL分类

  1. DDL：数据定义语言，用于定义数据库对象（数据库，表，字段）
  2. DML：数据操作语言，用于对数据库表中的数据进行增删该
  3. DQL：数据查询语言，用于查询数据库中表的结构
  4. DCL：数据控制语言，用于创建数据库用书，控制数据库的访问权限

DDL

库操作

查询所有数据库：

show databases;

查询当前数据库：

select database();

创建：

create database [if not exists] 数据库名 [default charset 字符集] [collate 排序规则];

删除：

drop database [if exists]数据库名；

使用：

use 数据库名；

表操作

查询

查询当前数据库所有表:

show tables;

查询表结构:

desc 表名;

查询指定表的建表语句：

show create table 表名;
创建
create table 表名(
    字段1 字段1类型[comment 字段1注释],
    字段2 字段2类型[comment 字段2注释],
    字段3 字段3类型[comment 字段3注释],
    ......
    字段n 字段n类型[comment 字段n注释]
)[comment 表注释];
案例

案例：在itheima库中创建tb_user的表，包含id，name，age,gender,并查询该表结构。

create datebase itheima;
use itheima;
create table tb_user(
    id int comment '编号',
    name varchar(50) comment '姓名',
    age int comment '年龄',
    gender varchar(1) comment '性别'
)comment '用户表';
desc tb_user;
修改

添加字段

alter table 表名 add 字段名 类型（长度） [comment 注释];

案例：为emp表添加一个新的字段“昵称”为nickname，类型为varchar（20）

desc cmp;
alter table emp add nickname varchar(20) comment "昵称";

修改数据类型

alter table 表名 modify 字段名 新数据类型；

修改字段名和字段类型

alter table 表名 change 旧字段名 新字段名 数据类型 [comment 注释];

案例：将emp中的nickname改为username，类型为varchar(30)

alter table emp change nickname username varchar(30) comment "昵称";

删除字段

alter 表名 drop 字段名;

案例：将emp表中的username删除

alter emp drop username;

修改表名:

alter table 表名 rename to 新表名;

案例：将emp表名该为employee

alter username rename to employee;

删除表:

drop table [if exists] 表名;

删除指定表，并重新建立该表：

    truncate table 表名;

案例：将表tb_user删除

drop table if exists tb_user;

DML

添加数据

给指定字段添加数据：
insert into 表名(字段名1，字段名2，...)values(值1，值2);

案例：

insert into employee(id, workno, name, gender, age, idcard, entrydate) values("1","1","黑马","男",18,"123456789012345671","2024-3-28");
给指定字段添加数据：
insert into 表名 values(值1，值2);

案例：

insert into employee values (2,"2","马黑","女",81,"012345678901234562","2024-3-28");
批量添加数据
insert into 表名(字段名1，字段名2,...) values(值1，值2，...),(值1，值2,....),(值1，值2，...);

insert into 表名 values(值1，值2,....),(值1,值2,...),(值1,值2,...);

案例：

insert into employee values (3,"3","小黑马","男",8,"012345678901234563","2024-3-28"),(4,"4","中黑马","男",18,"012345678901234564","2024-3-28");

修改数据

update 表名 set 字段1=值1,字段2=值2,....[where 条件];

案例：将编号为1的name修改城itheima：

update employee set name="itheima" where id=1;
